Our Method

1. Experimental Setup

We administer a 20 minute visuospatial working memory task, which is used to assess attentional performance.

2. Detailed Eye Biometrics

We record eye biometrics as a patient completes a task, looking at pupillometrics and eye gaze directions, which are then analyzed using our algorithms.

3. Machine/Deep Learning Backend

Biometric data is sent to a backend server and inputted into an optimized machine learning algorithm to analyze the data.

4. Exploratory Platform

A summary of eye biometrics is presented, along with detailed statistics and a probability of a patient having ADHD.
